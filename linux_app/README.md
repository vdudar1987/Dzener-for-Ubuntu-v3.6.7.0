# Dzener Linux Edition v3.7.0

Независимая Linux-версия приложения `Dzener`, обновлённая и расширенная для локального использования.

## Что нового

- улучшена валидация URL (`https://dzen.ru/...` и `https://zen.yandex.ru/...`);
- добавлена устойчивость к повреждённому `state.json`;
- у задач появились временные метки `created_at`;
- в CLI добавлены команды:
  - удаление своих задач;
  - экспорт списка задач в JSON;
  - сброс баллов и статистики;
- добавлены аргументы командной строки:
  - `--process N` для пакетной обработки;
  - `--version` для вывода версии;
- GUI обновлён: экспорт задач и расширенная информация о материалах.

## Возможности

- добавлять ссылки на публикации/видео;
- задавать тип контента и целевые действия;
- обрабатывать очередь задач обмена активностью;
- получать баллы (`+2` за каждое выполненное задание);
- хранить состояние в `~/.dzener-linux/state.json`;
- экспортировать задачи в `~/.dzener-linux/tasks_export.json`.

## Запуск

```bash
# консольный режим
python3 linux_app/dzener_linux.py

# графический интерфейс (Tkinter)
python3 linux_app/dzener_linux.py --gui

# пакетная обработка N заданий
python3 linux_app/dzener_linux.py --process 12

# версия
python3 linux_app/dzener_linux.py --version
```

## Примечание

Это локальная Linux-реализация, не использующая Windows-бинарники (`.dll`, `.exe`) из папки `windows/`.
