# Dzener Linux Edition

Независимая Linux-версия приложения `Dzener`, созданная на основе описания из этого репозитория.

## Что умеет

- добавлять ссылки на публикации/видео;
- задавать тип контента и целевые действия;
- обрабатывать очередь задач обмена активностью;
- начислять внутренние баллы (`+2` за каждое выполненное задание);
- хранить состояние в `~/.dzener-linux/state.json`.

## Запуск

```bash
# консольный режим
python3 linux_app/dzener_linux.py

# графический интерфейс (Tkinter)
python3 linux_app/dzener_linux.py --gui
```

## Примечание

Это локальная Linux-реализация, не использующая Windows-бинарники (`.dll`, `.exe`) из папки `windows/`.
